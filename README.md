# NexShop AI Cloud Platform 🚀

An AI-powered cloud-native E-commerce platform deployed on AWS using a microservice architecture.

## Overview

This project combines cloud computing, AI semantic search, Docker containerization, and scalable backend services to build a production-style E-commerce platform.

## Features

- AI-powered semantic product search
- Dockerized microservices architecture
- AWS cloud deployment
- FastAPI AI service
- NestJS backend
- React frontend
- PostgreSQL + pgvector vector database
- AWS SageMaker integration
- Docker bridge networking
- Production-grade debugging and deployment

## Cloud Architecture

User Browser → AWS S3 → AWS EC2 Backend → Docker Network → FastAPI AI Service → AWS SageMaker → AWS RDS PostgreSQL + pgvector
